SACC / SCSS SAMPLE PAGE

SACC / SCSS (サス) とは
・SASS【 Syntactically Awesome Stylesheets 】
CSS の、コーディング効率やソースコードの保守性を向上させる技術が、Sass です。
・SCSS【 Sassy CSS 】
SASSの別文法（Sassy CSS）CSS風にかける文法。
どんなことができる？
・変数　（$var）
・セレクタのネスト
・ミックスイン（@mixin）
・セレクタの継承　（@extend）
  • 1) 環境構築（rubyのインストール、コンパイラのインストール）
  • 2) テキストエディタなどでsass(またはscss)でcssを記述。　※拡張子はsassまたはscss
  • 3) コンパイルする。　※コマンドで行うこともあるが今回はコンパイラを使用
